# CryptoVote - Güvenli Elektronik Oylama Sistemi

## Proje Hakkında

CryptoVote, Python programlama dili kullanılarak geliştirilen güvenli bir elektronik oylama sistemidir. Proje; seçmen doğrulama, oy bütünlüğünün korunması ve güvenli veri yönetimi prensipleri esas alınarak geliştirilmiştir.

Bu çalışma, güvenli yazılım geliştirme süreçlerini uygulamalı olarak deneyimlemek ve elektronik oylama sistemlerinde karşılaşılabilecek temel güvenlik gereksinimlerini incelemek amacıyla hazırlanmıştır.

---

## Özellikler

* Seçmen doğrulama mekanizması
* Token tabanlı oy kullanımı
* Oy bütünlüğünün korunması
* JSON tabanlı veri saklama
* SHA-256 ile veri bütünlüğü kontrolü
* Tekrarlı oy kullanımını önlemeye yönelik kontroller
* Git tabanlı sürüm yönetimi

---

## Kullanılan Teknolojiler

* Python
* JSON
* Git
* SHA-256
